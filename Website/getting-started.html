Getting Started

Follow the instructions below to get Glasstodon up and running.

Wi-Fi Setup

You'll need an internet connection to use the app. If you don't already have one, you can set it up by clicking the below button.

Installation

Since Google shut down all Glass-related services a while ago, you'll need to sideload the app using ADB. Don't worry, it's pretty straightforward!

  1. Download the latest APK from the releases page.
  2. Make sure you have ADB installed on your computer. This article from XDA should get you on the right track.
  3. Connect your Glass to your computer using a USB cable.
  4. Open a terminal and run adb install path/to/glasstodon.apk.
  5. Once the installation is complete, you should see the app in your Glass menu. Open it and follow the login instructions.

Logging in

Enter the domain of your Mastodon instance (e.g. mastodon.social).
